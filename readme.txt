== Description ==
Rewrites and redirect all external links in posts, comments and author links with info page.
External links will redirected over a special info page with information about leaving the blog.

**Usage**
After activation it's ready to work.
You can setup by navigating admin menu to "WP Js External Link Info".

== Installation ==

1. Download and unzip the last version of this plugin.
2. Upload the wp-js-external-link-info folder to ./wp-content/plugins/
3. Go to WP Admin panel > Plugins > activate "WP Js External Link Info"
